## Getting the Code

```sh
  git clone https://github.com/oscarmarina/blockquote-playground-mixin-signals.git
  cd blockquote-playground-mixin-signals
```

### Installation

```js
npm i && npm start
```

### Serve with `vite`

```js
  npm start
```

### Serve Import Maps (no builds required)

```sh
  python -m SimpleHTTPServer 8000
```
